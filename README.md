# SceneHelper

### 在这个快节奏的虚拟创作时代，每一个灵感迸发的瞬间都显得尤为珍贵。作为经常进行虚拟主播切片或VR摄影的创作者，您是否曾为捕捉到一个完美的Punchline或绝美场景而兴奋，却又在后续的素材整理中懊恼无比？

了解制片流程的朋友应该也明白，有专门记录镜头情况的工种叫做场记。需求与概念合二为一，经过简单的操作，数字场记板诞生了。
![preview1.jpg](https://github.com/Bigdangg/SceneHelper/blob/main/previewpic/preview1.jpg)
通过OBS的WebSocket功能，数字场记板实现了与OBS的无缝对接。无需再手动切换多个界面，您可以轻松控制OBS的录制情况，一切操作尽在指尖掌控。
![preview2.jpg](https://github.com/Bigdangg/SceneHelper/blob/main/previewpic/preview2.jpg)
只需轻轻一点，数字场记板便会自动记录下录制的时间和状态，并生成详细的Excel文档。时间与状态一目了然，只为让后期选片工作更加自如。
